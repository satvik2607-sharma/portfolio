My Projects

Women Security App

The Women Security App is a simple yet powerful Android application designed to enhance women's safety. The app utilizes the latest Flutter and Dart technology, providing a user-friendly interface that is easy to navigate. The app is a team project that involves six dedicated members, each contributing unique skills to ensure its success. My role in the project was as a front-end developer, where I focused on creating the Tip of the Day page that is easily accessible from the navigation bar. I am proud to have been a part of this project, and I believe that it will help make a significant impact in improving women's security.

To-do List

This project is a To-do list app designed to help users increase their productivity by organizing their tasks in a simple and effective manner. The app provides a platform for users to note down and prioritize their daily tasks, set reminders, and mark tasks as complete. Developed using Flutter, Dart, and Firebase technologies, the app offers a seamless and efficient user experience. As a member of the development team, my role included contributing to the front-end development and collaborating with team members to ensure the successful delivery of the project.

Investment IQ

Investment IQ is a start to end one stop solution for investing in stock markets. It is a flutter based web application, that is built on the Django REST framework, and uses an LSTM built with Tensorflow for predicting the stock price using price action. It uses the Breeze API by ICICI Direct to perform seamless and secure transaction of assets between the application and the demat account.
